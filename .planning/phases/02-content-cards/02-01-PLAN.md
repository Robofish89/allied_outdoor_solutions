<task type="auto">
  <name>Task 1: Create UseCaseCard component</name>
  <files>src/components/cards/UseCaseCard.tsx</files>
  <action>
Create a card component that accepts a `UseCase` object and renders:

1. **Header area**: Title (serif, large) + subtitle (sans, muted) + timeline badge (gold pill with timelineLabel text)
2. **Description**: Full description paragraph with Allied-specific context. Show `description` as the main body, then `alliedContext` as a second paragraph or distinct callout with slightly different styling (e.g., italic or left-bordered).
3. **Process flow visual**: A horizontal 3-step flow showing processSteps — input → AI → output. Use icons or colored dots for step types: input (allied-slate), AI (allied-gold), output (allied-navy). Each step shows its label. Connect steps with arrows or lines. This is the static version — Phase 3 adds animation.
4. **Tools/tech badges**: Row of pill badges showing each tool from the `tools` array. Use small rounded pills with subtle background (e.g., allied-navy/5 bg, allied-navy text).
5. **ROI estimate**: Display `roi` text with a small icon or label prefix like "Expected Impact:" in gold.

**Styling guidelines:**
- Card has white bg, rounded-2xl, subtle shadow (shadow-sm), generous padding (p-6 md:p-8)
- Use the existing design token colors from globals.css (allied-navy, allied-gold, allied-cream, allied-slate, allied-charcoal)
- Headings use serif font (inherited from globals.css h1-h6 rule)
- Body text uses sans font
- Responsive: Single column stack on mobile, comfortable reading on desktop
- Do NOT add hover effects or transitions — Phase 3 handles animation
- Do NOT use any external icon library — use simple SVG arrows for the process flow or Unicode characters

**Process flow layout:**
- On desktop: horizontal row with 3 steps connected by arrows
- On mobile: can remain horizontal but steps stack labels below icons if needed
- Step type styling: input gets a blue-ish dot/icon, AI gets gold, output gets navy
- Arrow between steps: simple SVG `→` or a thin line with arrowhead
